# Telegram Bot

Telegram Bot service scaffold based on [grammY][1] & [Node.js][2] & [TypeScript][3]

## Development

### Installation

```shell
npm i pnpm -g
pnpm i
```

### Start Development environment

```shell
pnpm dev
```

[1]: https://grammy.dev/
[2]: https://nodejs.org/
[3]: https://www.typescriptlang.org/
